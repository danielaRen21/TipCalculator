Interface

La interfaz de la Calculadora de Propinas y Consumos está diseñada para ser clara, intuitiva y responsive. Permite al usuario registrar consumos de manera sencilla y seleccionar el porcentaje de propina en tiempo real, mostrando el total de forma inmediata.

El proyecto utiliza React con Vite para un rendimiento óptimo, TypeScript para mayor seguridad en el manejo de datos y Tailwind CSS para un diseño moderno y consistente. La experiencia está pensada para ser fluida, accesible y adaptable a dispositivos móviles y de escritorio.
